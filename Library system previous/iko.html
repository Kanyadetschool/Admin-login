Student Portal Dashboard

This is a responsive student portal navbar with a hamburger menu for mobile devices. Try resizing your browser window or opening this on a mobile device to see the hamburger menu in action!

Features:

  • Smooth hamburger menu animation
  • Responsive design that adapts to different screen sizes
  • Student info display in both desktop and mobile views
  • Active state management for navigation links
  • Gradient background with modern styling
  • Bootstrap Icons integration
